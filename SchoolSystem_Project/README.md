# Digital Edu Pro School System

Ongoing personal project focused on building a fully customizable school operations platform for international and private schools.

## Folder Contents

- `index.html`: visual project brief page
- `FEATURES.md`: complete feature list and module scope
- `screenshots/`: current UI screenshots (login, dashboards, ICT screens)

## Project Focus

- Centralize academic setup, grading, reporting, and attendance.
- Support role-specific workflows for Admin, ICT, TA, HRT, and Subject Teacher users.
- Provide flexible configuration to match each school's curriculum design and policies.
